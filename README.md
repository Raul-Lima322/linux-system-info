Um script simples e eficiente em Bash para monitorar e exibir informações essenciais do seu sistema Linux diretamente no terminal.

## Como Rodar

Para rodar o programa, basta digitar:

`bash main.sh`

Ou, se preferir tornar o script um executável:

`chmod +x main.sh`  
`./main.sh`

---

**Obs:** * É necessária a ferramenta `curl` instalada (para obter o IP externo).
* O script funciona exclusivamente em Sistemas Operacionais baseados em Linux.
